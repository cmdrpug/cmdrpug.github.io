MACHINE LEARNING GROUP PROJECT

This is the only project I worked on thats hosted on github as of now. It was a final project for a machine learning course I took last year. The project is based on Conway's game of life which follows a grid layout where each space is either on or off. The state of adjacent spaces changes how a space will act on the next turn. Our goal was to use multiple ML techniques to try and find stable seeds for the game. I worked on a neural network approach.
